In this project i will send and receive text using nrf24l01 module.
There will be two code one for transmitter and another for receiver.

Receiver
--------
The receiver will show received text in the serial monitor.


Transmitter 
-----------
To transmit data,we will have to enter our text in the serial monitor of the transmitter .  Then if we press send or 
press enter the text will be transmitted and shown on the serial monitor of the receiver


This codes use RF24 library.
Library can be found in: TMRh20/RF24, https://github.com/tmrh20/RF24/
